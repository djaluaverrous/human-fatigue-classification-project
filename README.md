# 🧠 Human Fatigue Prediction System

AI-powered web app untuk memprediksi tingkat kelelahan manusia 
menggunakan Random Forest Classifier.

## 🚀 Live Demo
[Click here](https://your-app-name.streamlit.app)

## 📊 Features
- EDA Dashboard interaktif
- Prediksi fatigue level real-time
- Batch prediction via CSV upload
- Model performance comparison
- AI Recommendation system

## 🛠️ Tech Stack
- Python, Streamlit, Scikit-Learn, Plotly, Pandas

## ▶️ Cara Menjalankan Lokal
```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
```
